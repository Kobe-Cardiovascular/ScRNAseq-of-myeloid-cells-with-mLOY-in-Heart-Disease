# ScRNAseq-of-myeloid-cells-with-mLOY-in-Heart-Disease

We analyzed single-cell RNA sequencing (scRNAseq) data from ten male patients with AF in our own scRNAseq dataset (GSE261170) and publicly available datasets of left atrial tissue and right atrial appendage tissue (GSE224959, GSE263154) and six male patients with DCM in publicly available datasets of left ventricular cardiac tissue and right ventricular cardiac tissue (GSE183852, GSE145154). Myeloid cells were extracted from twelve sample of only male patients in these datasets, and comparative analyses were performed. mLOY cells were defined as cells showing the combinatorial absence of expression of all Y chromosome derived genes.

The code is written in R. The R code uses the Seurat package.
